## IP Manipulator v2

Alternate IP Address notations. I wrote a terrible version 
years ago while trying to convince myself I can tolerate 
python. Rather than fix it in a language I never care to use,
here's a C++ rewrite. 

##### examples:

$ ./IP_Manipulator 192.168.1.1
IP Manipulator
--------------
192.168.1.1 --> 192.168.1.1  :: (Decimal)
192.168.1.1 --> 192.168.257  :: (Decimal)
192.168.1.1 --> 192.11010305  :: (Decimal)
192.168.1.1 --> 3232235777  :: (Decimal)
192.168.1.1 --> 0xc0.0xa8.0x1.0x1  :: (Hex)
192.168.1.1 --> 0xc0.0xa8.0x101  :: (Hex)
192.168.1.1 --> 0xc0.0xa80101  :: (Hex)
192.168.1.1 --> 0xc0a80101  :: (Hex)
192.168.1.1 --> 0300.0250.01.01  :: (Octal)
192.168.1.1 --> 0300.0250.0401  :: (Octal)
192.168.1.1 --> 0300.052000401  :: (Octal)
192.168.1.1 --> 030052000401  :: (Octal)
